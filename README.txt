This repository stores data set information for use in academic works.

Dataset.txt - Contains a list of hashes of malicious binaries. The first ~400 entries are grouped as Dataset 1. All the hashes in the file (including the first ~400) are considered Dataset 2.

Labels_from_VirusTotal.txt - Contains all the labels from these hashes according to VirusTotal. This is a set of classifications and labels for each hash in the Datasets.txt file.

To cite this dataset, please correctly contact the owner of this repository on github (dtgw).
